# 🛒 Aplikasi Kasir Pintar

Aplikasi Kasir (Point of Sale) berbasis Android untuk memudahkan pengelolaan barang, transaksi belanja, pencatatan hutang, hingga pencetakan struk melalui printer thermal Bluetooth.

## ✨ Fitur Utama
- **Kasir & Keranjang Belanja:** Perhitungan total belanja secara otomatis.
- **Manajemen Gudang:** Tambah, edit, hapus, dan pantau stok barang secara *real-time*.
- **Pencatatan Hutang:** Fitur khusus untuk menyimpan dan memantau transaksi pelanggan yang belum lunas (Hutang).
- **Riwayat Transaksi:** Tinjau kembali detail penjualan dan bukti pembayaran di masa lalu.
- **Cetak Struk Bluetooth:** Terintegrasi dengan printer thermal bluetooth (ESC/POS) untuk mencetak struk fisik.
- **Penyimpanan Cloud:** Data tersimpan aman menggunakan Firebase Realtime Database.

## 🚀 Cara Instalasi (Untuk Customer)
Untuk mengunduh dan memasang aplikasi ini di HP Android Anda, ikuti langkah berikut:

1. Buka menu **[Releases](../../releases)** di halaman ini.
2. Unduh file **`.apk`** pada versi rilis terbaru.
3. Buka file APK yang sudah diunduh di HP Anda.
4. Jika muncul peringatan keamanan, masuk ke Pengaturan HP dan aktifkan **"Izinkan instalasi dari Sumber Tidak Dikenal"** (*Install from Unknown Sources*).
5. Lanjutkan proses instalasi hingga selesai, dan aplikasi siap digunakan!

*Dibuat menggunakan Android Studio (Kotlin) & Firebase.*
